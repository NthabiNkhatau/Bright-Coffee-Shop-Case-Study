# Bright-Coffee-Shop-Case-Study
This repository contains my full analytical solution to the Bright Coffee Shop Sales Analysis Case Study, where I explore customer purchase behaviour, product performance, and operational trends using SQL, Excel, and visual planning tools.
The goal is to equip a newly appointed CEO with actionable, data-driven insights that can increase revenue, optimize product offerings, and enhance store performance.
# Project Overview
Bright Coffee Shop provided a dataset containing daily transaction-level sales data, including product details, store information, pricing, quantities, dates, and times.
As a Junior Data Analyst, my role was to:
Understand performance across products, time periods, and store segments
Identify high and low revenue drivers
Discover patterns in customer spending behaviour
Build dashboards and clear recommendations for executive decisiom
